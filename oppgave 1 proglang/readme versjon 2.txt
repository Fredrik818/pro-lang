Kontrollspørsmål


Her har vi tre filer, CSS, JS og HTML. 
Modellen er Html filen, siden det er denne filen som er modellen og holder på dataen vi har. 
Utseende er CSS Fila, denne fila bestemmer hvordan vår data skal vises. 
Kontrolløren er JS fila. Det er den som kan kontrollere hva som skal vises av informasjonen i HTML fila.
Hele poenget er å minske repetisjon av data, og gjøre det mer oversiktlig.
