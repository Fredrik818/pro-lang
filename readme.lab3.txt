SPØRSMÅL LAB 2

0) Hva er fordelen med et eksisterende JavaScript bibliotek som jQuery i motsetning til å
skrive en “rå” JavaScript for dynamisk endring av HTML sider?

		- Fordelen med jQuery er at man slipper å skrive så mye. Jquery gjør utvikling av websider enklere, og det tar kortere tid
		  å lage dem. Jquery inneholder mange ferdiglagde komponenter som man enkelt kan benytte seg av, istedet for å kode alt "fra bunnen av". 


1) Les denne artikkelen
http://www.ibm.com/developerworks/web/library/wa-htmlmark/index.html og svar på følgende
spørsmål: Hva er sterke sider til HTML/CSS og hva er sterke sider til Canvas? I hvilke
situasjoner er det meningsfult å bruke Canvas?

		- Sterke sider med HTML/CSS er at det er enkelt å skirve og endre på tekststørrelse/skriftfarge/skrifttype. I CSS kan man enkelt koble til et stilsett 
		  som gjør at man kan velge en stil som gjelder for hele dokumentet, eller flere dokumenter. Man sparer dermed både plass og tid, koden ser bedre ut, og det er
          også flere muligheter enn i HTML. I canvas kan man også velge tekststørrelse/skriftfarge/skrifttype, men på en litt annen måte. 
		  Canvas brukes gjerne til å tegne grafikk eller lage animasjoner i Javascript. 


2) Hvilken grafisk presentasjonsmåte (se Innledning til denne oppgaven) hadde du sagt at
HTML5 Canvas representerer? Forklar.


		- Canvas er best egnet for presentasjonsmåter hvor det er nødvendig å oppdatere skjermen fort (realtidapplikasjoner).
		  Det er vanlig i bruk av spill og bruk av rastergrafikk hvor det er bruk av piksler og ikke abstrakte strukturer (komponenter).
